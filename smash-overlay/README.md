#### smash-overlay

---

Smashes a `png` overlay atop a set of images. Takes images in `/original` and applies `overlay.png`, then saves in `/composed`. Overlay and original images should be the same size.

Open `smash.command` to start.

```
├── smash-overlay/
│   ├── original/
│   ├── composed/
│   overlay.png
│   smash.command
```
